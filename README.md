# Workout Companion

A polished, offline-friendly workout follow-along page for a 30-minute equipment-free home session.

Open `index.html` in Chrome, go full-screen, press Start, and follow the timer. The app guides you through warm-up, main circuit, finisher, and cooldown phases with large readable prompts, inline exercise illustrations, progress tracking, technique tips, and Web Audio sound cues.

## Highlights

- Single-file HTML app with embedded CSS and JavaScript
- No backend, build tools, package install, or external assets
- Desktop full-screen first, with responsive mobile support
- Dark shadcn/ui-inspired dashboard styling
- Large central interval timer and total workout countdown
- Current phase, interval type, round number, next exercise, and progress bar
- Start, pause, resume, restart, skip next, and previous controls
- Inline SVG/CSS exercise illustrations, so the page works offline
- Rotating motivation panel and exercise-specific technique tips
- Web Audio API cues for work intervals, rest/recovery, countdown, phase changes, and completion
- Mute/unmute button and volume slider

## Workout Plan

### Warm-up: 5 minutes

- March in place / light jogging
- Arm circles + shoulder rolls
- Hip circles
- Bodyweight good mornings
- Easy squats
- Easy push-up warm-up

### Main Circuit: 18 minutes

Three rounds of:

- Squats
- Push-ups
- Reverse lunges
- Plank shoulder taps
- Glute bridges
- Mountain climbers

Each movement uses a 40-second work interval followed by a 20-second rest or recovery interval.

### Finisher: 4 minutes

EMOM style:

- 8 burpees

Alternatives:

- 12 squat thrusts
- 10 fast squats + 10 mountain climbers

### Cooldown: 3 minutes

- Child's pose
- Hip flexor stretch right
- Hip flexor stretch left
- Hamstring stretch
- Chest / shoulder stretch

## How To Use

1. Open `index.html` in Chrome.
2. Put the browser in full-screen mode.
3. Press Start.
4. Adjust volume or mute from the control bar.
5. Follow the current exercise, timer, and next-exercise prompts.

Sound is initialized only after Start is pressed, which keeps it compatible with browser autoplay rules.

## Editing The Workout

The workout is stored as a JavaScript array of interval objects generated by `buildWorkout()` inside `index.html`.

To customize it:

- Change exercise names in `buildWorkout()`
- Adjust interval durations with the `duration` field
- Update technique tips in the `tips` object
- Change phase colors in `phaseColors`
- Edit or add inline SVG poses in `visualSvg()`

Because everything is in one file, changes can be tested immediately by refreshing the browser.
